# Using the past to constrain the future: how the palaeorecord can improve estimates of global warming

## Question

What are the key takeaways? Make it a downloadable Md file

## Answer

The paper discusses the use of palaeoclimate records to improve estimates of climate sensitivity, which is crucial for predicting global warming. It highlights the persistent uncertainties in climate sensitivity estimates and suggests that using geological data can help reduce these uncertainties. The authors review methods that combine instrumental climate observations with palaeoclimate data to better constrain climate sensitivity estimates.
