# Machine Learning and Citizen Science Approaches for Monitoring the Changing Environment

## Response

The methods described in the paper involve the use of mobile applications and a 3-tier architecture for data collection and analysis. Specifically, the study deployed and tested applications for monitoring Hurricane Sally and Hurricane Zeta, utilizing features such as Filter ComboBox, ListModel, and ListElement in the application interface ("Results From The First Chapter", page 39). Additionally, the study employed a 3-tier architecture where volunteers collected field observations using an application. This architecture involved a presentation tier for user interaction, an application tier for processing, and a data tier for storage and retrieval ("Related Work", page 34).
